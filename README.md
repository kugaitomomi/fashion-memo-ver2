## Getting Started

Get a local copy up and follow these simple steps.

### Prerequisites

To use this software you will need npm. To install run

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

_After installing npm_

1. Clone the repo
   SSH
   ```sh
   git clone git@github.com:kugaitomomi/fashion-memo-ver2.git
   ```
   HTTPS
   ```sh
   git clone https://github.com/kugaitomomi/fashion-memo-ver2.git
   ```
2. Install NPM packages
   ```sh
   cd server
   npm install
   cd ../client
   npm install
   ```
3. Run express app in /server/
   ```js
   npm run dev
   ```
   And react in /client/
   ```js
   npm run start
   ```
